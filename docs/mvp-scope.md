# MVP Scope

## Included
- Homepage
- Search page with filters
- Listing detail page
- Compare page
- Saved listings
- Saved searches
- Inquiry flow
- Admin dashboard
- Listing moderation
- Area pages
- Seed data

## Excluded
- Payments
- Mobile app
- Full AI features
- Chat system
- Scraping pipelines
- Advanced analytics
- Mortgage integrations

## Target users
1. Renters in Greater Colombo
2. First-time buyers

## Success criteria
- User can search and find relevant listings
- User can save and compare listings
- User can contact the listing owner or agent
- Admin can control listing quality

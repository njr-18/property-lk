# API Contracts

## Public web

- `GET /api/search`: search results and facets
- `GET /api/listings`: listing collection retrieval
- `GET /api/listings/[id]`: listing detail retrieval
- `POST /api/inquiries`: capture inquiry intent
- `GET|POST /api/saved-listings`: manage saved listings
- `GET|POST /api/saved-searches`: manage saved searches

## Admin

- `GET /api/moderation`: moderation queue summary
- `GET /api/duplicates`: duplicate review queue
- `GET /api/verification`: verification queue

# Schema Overview

Core entities:
- User
- Profile
- Location
- Listing
- ListingImage
- ListingFeature
- SavedListing
- SavedSearch
- Inquiry
- ListingVerification
- DuplicateCluster
- Report
- AreaPage
- Guide

Key principles:
- Listings must be searchable by location, price, and type
- Locations must be normalized for Sri Lankan area naming
- Listings must support trust signals and moderation
- Duplicate detection must be supported
- Admin review must be possible
